Self-Hosted Homelab Scripts

Four production-grade, zero-dependency Python scripts to run a Raspberry Pi as a private VPN, ad-blocker, and live monitor. Free and open. No pip. No Flask. Just runs.

What's inside (all free)

  • pihole_blocklist.py — pushes 25 hand-picked tracker domains to Pi-hole v6 + rebuilds gravity
  • dashboard.py — live system monitor (CPU/mem/disk/temp) as JSON API + HTML page, no Flask
  • wg_peer.py — onboards a phone to WireGuard in 30s (writes .conf, prints QR)
  • cron_task.py — scaffolds correct systemd timers from a template
  • Every script commented line-by-line, self-documenting (--help)
  • Pure Python standard library — runs on stock Raspberry Pi OS / Ubuntu
